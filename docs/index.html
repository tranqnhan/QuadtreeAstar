Quadtree A*

https://github.com/tranqnhan/QuadtreeAstar
Controls Left click - Draw/erase on canvas E - Toggle erase/draw A - Increase brush size S - Decrease brush size
Note: Right now, it only build quadtree and graph. No A* yet.
Technologies: C++, Raylib, Emscripten, WASM, Ankerl's Unordered Dense
References:
  • K. Aizawa and S. Tanaka, "A Constant-Time Algorithm for Finding Neighbors in Quadtrees," in IEEE Transactions on Pattern Analysis and Machine Intelligence, vol. 31, no. 7, pp. 1178-1183, July 2009, doi: 10.1109/TPAMI.2008.145.
